# Gym App 

### UI
```bash
cd ui

# Setup
npm install
npm run ios
```

### API
Provide `ATLAS_URI` in a new file called `config.env` in `/api`:
```bash
ATLAS_URI=
PORT=5000
```

Useful commands:
```bash
cd api

# Setup
npm install
npm run start

# Running tests
npm run test

# Updating swagger
npm run swagger
```
