# Gym App 

![gym app logo](ui/logo.png)

### Database
Provide MongoDB `ATLAS_URI` in a new file called `config.env` in `/api`:
```bash
ATLAS_URI=
PORT=5000
```
### API
```bash
cd api

# Setup
npm install
npm run start

# Running tests
npm run test

# Updating swagger
npm run swagger
```

### UI
```bash
cd ui

# Setup
npm install
npm run ios
```
